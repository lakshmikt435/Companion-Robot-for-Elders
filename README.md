# Companion-Robot-for-Elders

As part of my final year project , we developed "A Companion Robot for Elders ", aimed at enhancing the quality of life for senior citizens by providing emotional support , friendship and socialization. 

Built on a Raspberry Pi 4 and integrated with modules like servo motors, speakers, microphone, camera, and LCD display, this robot was designed to:
🔹 Provide timely wake-up calls , hydration and medication reminders
🔹 Share live news, weather updates, and inspirational quotes
🔹 Encourage light exercise and offer interactive storytelling
🔹 Reduce loneliness through intelligent, conversational companionship powered by Gemini AI

Key Features:
🗣 Voice interaction using speech-to-text and text-to-speech
🖼 LCD facial expression display
🎥 Live video feed via Pi camera
🦾 Gesture response with servo motors
🎶 Audio input/output with 35mm mic and speakers
💻 Designed using Fusion 360 and coded with Python
🎥 Include movement capabilities using line follower robotic chassis powered on an arduino



In today's fast-paced world , where everything is digitally driven , this solution explores how low-cost, interactive robots can bridge the gap between independence and assisted living.
